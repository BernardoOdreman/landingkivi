Aceleradores Debajo del Volante

Sistemas Kivi Easy-Fit: K4, K5 y K6 para una conducción adaptada y segura

Kivi K5 Easy-Fit

El acelerador K5 Easy-Fit es la solución perfecta para personas con movilidad reducida en las piernas, permitiendo controlar el acelerador con la mano derecha sin esfuerzo. Se instala debajo del volante e integrado en el habitáculo, con tres modos de uso:

  • Tirando del acelerador hacia el volante
  • Pulsando con el dorso de los dedos
  • Accionando con el pulgar (mano libre)
  • Función ECO: reduce potencia al 60% para maniobras
  • Aumenta espacio entre piernas para mayor seguridad
  • Compatible con volantes deportivos
Solicitar información

Kivi K6 Easy-Fit

El acelerador K6 Easy-Fit ofrece una solución avanzada con mayor personalización y funciones adicionales para una experiencia de conducción adaptada más completa. Incluye función ECO para reducción de consumo.

  • Control de velocidad de crucero integrado
  • Posibilidad de conexión con sistemas multifunción
  • Materiales de alta resistencia y durabilidad
  • Sistema de seguridad con doble sensor
  • Compatible con vehículos eléctricos e híbridos
  • Diseño ergonómico que evita golpes involuntarios
Solicitar información

KS: Acelerador de Aro Sobre el Volante

El KS es el nuevo acelerador de aro sobre el volante que permite acelerar ejerciendo una ligera presión sobre el dispositivo.

  • Instalación sin cables
  • Disponible en versión fija o removible
  • Función ECO integrada para ahorro de combustible
  • Operación intuitiva con ligera presión manual
